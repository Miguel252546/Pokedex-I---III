# Pokédex - Generaciones I a III

Una Pokédex minimalista que muestra información de los Pokémon de las tres primeras generaciones (Kanto, Johto y Hoenn). El proyecto utiliza la [PokeAPI](https://pokeapi.co/) para obtener los datos de los Pokémon.

## Características

- Búsqueda de Pokémon por nombre
- Filtrado por generación (Kanto, Johto, Hoenn)
- Filtrado por tipo de Pokémon
- Diseño responsive
- Animaciones de Pokémon evolucionando
- Modal con detalles de cada Pokémon
- Interfaz accesible con ARIA labels

## Tecnologías utilizadas

- HTML5
- CSS3 (Animaciones y Grid)
- JavaScript vanilla
- [SweetAlert2](https://sweetalert2.github.io/) para notificaciones
- [PokeAPI](https://pokeapi.co/) como fuente de datos

## Cómo usar

1. Visita la [página del proyecto](https://tu-usuario.github.io/pokedex-page/)
2. Utiliza la barra de búsqueda para encontrar Pokémon por nombre
3. Usa los filtros para mostrar Pokémon por generación o tipo
4. Haz clic en cualquier Pokémon para ver más detalles

## Desarrollo local

1. Clona este repositorio
2. Abre el archivo `index.html` en tu navegador
3. ¡Listo!
